### 4. 涉及的编程接口 (API)
图片右下角列出了需要使用的核心 C 语言 Socket 函数：
*   `int socket(int protofamily, int type, int protocol)` - 创建套接字
*   `int listen(int sockfd, int backlog);` - 监听端口
*   `int connect(int sockfd, const struct sockaddr ...` - 发起连接
*   `int accept(int sockfd, struct sockaddr *addr ...` - 接受连接
*   `int inet_pton(int af, const char *src, void ...` - IP地址转换
*   `read`, `write`, `close` 函数 - 读写数据及关闭连接

### 5. 流程图逻辑 (左侧图示)
图片左侧展示了一个标准的 TCP Socket 通信流程：

**Server (服务器端):**
1.  **创建socket** -> `socket()`
2.  **绑定socket和端口** -> `bind()`
3.  **监听端口** -> `listen()`
4.  **接收来自客户端的连接请求** -> `accept()`
5.  **从socket中读取字符** -> `recv()`
6.  **关闭socket** -> `close()`

**Client (客户端):**
1.  **创建** -> `socket()`
2.  **连接指定计算机的端口** -> `connect()`
3.  **向socket发送信息** -> `send()`
4.  **关闭socket** -> `close()`
